#   BuildFlags: logidlelimit:36000

    构建在 10 小时内没有日志输出，终止构建进程

#   sandbox 沙箱

    用于隔离构建环境，以提高安全性和一致性，防止构建过程对宿主系统产生不良影响。
    常用的类型有：Docker、chroot、KVM、QEMU

#   Constraint  表示约束

    Constraint: sandbox docker      #构建任务须在Docker 容器内运行
    Constraint: hostlabel 9950X     #构建任务须在标记为 9950X 的构建节点上运行

    #overwrite参数表覆盖，overwrite:sandbox为覆盖sandbox的设置，设置构建任务将在 chroot 环境 中运行。
    Constraint: overwrite:sandbox chroot    
    
    #conditions是条件参数，conditions:package表示条件参数设置为包。
    #整体意思为：构建过程中llvm18 和 tensorflow 包，这些包的配置将会覆盖之前的配置。
    Constraint: overwrite:conditions:package llvm18 tensorflow

#   Macros 宏
        
共有两种设置宏的方式

##  %define定义

 用于prjconf文件本身内部，用于在RPM构建之前，计算spec文件的构建依赖关系。

##  Macros: 标签

会被 导出到构建环境中的 .rpmmacros 文件，会影响 spec 文件的处理过程。
        
        #宏的配置形式如下（需要放到Macros标签内）
        Macros:
        
        ...
        具体内容
        ...

        :Macros
